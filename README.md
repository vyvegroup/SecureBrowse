# SecureBrowse - Android Content Restriction App

A modern Android application built with **Kotlin** and **Material 3+** that enforces content restrictions with a professional, glassmorphism-based UI.

## Features

- **Material 3+ Design** - Latest Material Design components
- **Glassmorphism UI** - Glass reflection, gradient overlays, and modern visual effects
- **Content Filtering** - Restricts access to specified domains
- **Android 8.0+ Support** - Minimum API level 26
- **Smooth Animations** - Staggered entrance animations with physics-based interpolators
- **Edge-to-Edge Display** - Immersive full-screen experience

## Technical Stack

- **Language:** Kotlin
- **UI:** Material 3 (com.google.android.material)
- **Min SDK:** 26 (Android 8.0)
- **Target SDK:** 34 (Android 14)
- **Build:** Gradle 8.5 + AGP 8.2.2

## Build

```bash
./gradlew assembleRelease
```
